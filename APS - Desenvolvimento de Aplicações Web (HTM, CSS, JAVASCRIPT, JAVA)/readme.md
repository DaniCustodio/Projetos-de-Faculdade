## Objetivo	
O grupo deverá apresentar o projeto como um wireframe/mockup(rascunho) e criar uma plataforma 
de agenda de contatos geolocalizada, em que iremos integrar com a API do google maps.
As funcionalidades mínimas são: O usuário deverá se autenticar, cadastrar, 
alterar e excluir seus contatos, favoritar os contatos mais importantes e/ou recorrentes. 
No entanto, o grupo poderá adicionar outras funcionalidades diferenciadas, se assim desejar.
