**Prof:** Jorge Doria

**Disciplina:** Programação Orientada a Objetos

**Semestre:** 4ª Semestre - 2017.2

## OBJETIVO
O Banco Central, em seu portal de dados abertos, disponibiliza todas as informações 
sobre cotações diárias do Dólar comercial (venda e compra) em formato JSON no 
[LINK] (https://dadosabertos.bcb.gov.br/dataset/dolar-americano-usd-todos-os-boletins-diarios)

O trabalho consiste em desenvolver um aplicativo desktop que, acessando estes 
portais, exiba os preços de venda e de compra dos produtos cadastrados 
temporariamente em dólar.
